# 💣 Minesweeper (Java GUI)

A modern, feature-rich implementation of the classic Minesweeper game built completely from scratch using Java and Object-Oriented Programming (OOP) principles. 

## 🌟 Key Features
* **Dynamic Difficulty Levels:** Easy, Medium, Hard, Pro, and Extreme modes with adjustable grids.
* **Modern Dark/Light Mode:** Carefully designed GUI themes for a better visual experience.
* **Optimized Game Logic:** Fast and recursive zero-cell opening algorithm.
* **Interactive UI/UX:** Flagging system, real-time timer, and smooth mouse event handling.
* **Score System:** Top 10 leaderboard for each difficulty level saved in a local file.

## 🛠️ Technologies Used
* **Language:** Java
* **GUI Framework:** Java Swing / AWT
* **Architecture:** Core OOP (Inheritance, Encapsulation, Polymorphism)
